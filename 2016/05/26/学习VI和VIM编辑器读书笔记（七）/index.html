vi同类品的功能总览

在这里，我们只关注一些vim的功能主题。

多窗口编辑

缓冲区：保存了要编辑的文本。任何一个 文件都只有一个对应的缓冲区。
窗口：提供对缓冲区的观察窗口，让我们可以查看与修改缓冲区中的内容。

Exuberant ctags

Exuberant ctags插件
能够产生所有类型的c与c++语言标签
支持c与c++代码

新的tags文件格式

传统的tags文件有三个制表符分割的字段：
标签名称（一般是标识符）、包含标签的源文件、何处可找到标识符的指示符。
新的格式：
标签名称（一般是标识符）、包含标签的源文件、何处可找到标识符的指示符:;、描述属性的关键字：实际的值（可有多个，以tab分隔）
扩展的ctags关键字
关键字 意义
kind 单一字母，代表标签的词汇类型。f：函数、v：变量等
file 针对“静态”的标签，值为文件的名称。若值为空，则会以文件名字段为准。
function 针对本地的标签。其值是所定义的函数的名称。
struct 针对struct中的字段。其值是结构的名称。
enum 针对enum数据类型的值。其值为enum类型的名称。
class 针对c++的成员函数与变量。其值是类的名称。
scope 针对C++的类成员函数。（priviate）
arity 针对函数。定义参数的个数。

eg：
ARRAYMAXED awk.h 427;” d
AVG_CHAIN_MAX array.c 28;” d file:
array.c array.c 1;” F
第三个使用：tag array.c命令。或者将光标移动到文件名上，再使用^[命令移到该文件中。
